# StateMachineWatcher

**Inherits:** [Node](https://docs.godotengine.org/en/stable/classes/class_node.html)

Fires signals based on [StateMachine](../StateMachine) changes
## Description

The [StateMachineWatcher](../StateMachineWatcher) fires signals based changes to the given [StateMachine](../StateMachine). This enables other nodes to react to state changes.
