# BluetoothManager

**Inherits:** [Node](https://docs.godotengine.org/en/stable/classes/class_node.html)

BluetoothManager interfaces with the bluetooth system
## Description

This node is used to drive the bluetooth instance forward by calling its 'process()' method every frame to dispatch signals.
