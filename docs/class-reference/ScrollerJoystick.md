# ScrollerJoystick

**Inherits:** [Node](https://docs.godotengine.org/en/stable/classes/class_node.html)

Scroll a [ScrollContainer](https://docs.godotengine.org/en/stable/classes/class_scrollcontainer.html) using the right analog stick
## Description

Add as a child node to a [ScrollContainer](https://docs.godotengine.org/en/stable/classes/class_scrollcontainer.html) to allow controlling the scroll position using the right analog stick.
