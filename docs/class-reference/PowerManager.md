# PowerManager

**Inherits:** [Node](https://docs.godotengine.org/en/stable/classes/class_node.html)

Manages power settings.
## Description

The [PowerManager](../PowerManager) class is responsible for loading a [UPowerInstance](../UPowerInstance) and calling its 'process()' method each frame.
